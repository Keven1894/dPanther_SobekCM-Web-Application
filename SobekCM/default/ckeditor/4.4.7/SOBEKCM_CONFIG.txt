Creation instructions ( within CKBuilder )
---------------------------------------------

Start with CKEditor - Standard Package ( Version 4.4.7 )

Add the following plugins:
	- Auto Grow
	- Code Snippet
	- CodeMirror (Source) Syntax Highlighting
	- Div Editing Area
	- Find / Replace
	- Image BrowseR
	- Slideshow
	- Table Resize

Use Moona Color as the skin

For languages, add German, Spanish, French, and Dutch
